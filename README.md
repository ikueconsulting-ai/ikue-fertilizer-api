# IKUE Fertilizer API

Simple fertilizer price API for UREA, DAP, and NPK.
Used by IKUE AGROTECH for global price comparison.

## Endpoints

- /api/urea
- /api/dap
- /api/npk

Each endpoint returns:

{
  "commodity": "urea",
  "region": "global",
  "priceTon": 420,
  "priceKg": 0.42,
  "updated": "2026-06-16T00:00:00.000Z"
}
